    2.What is css box model?

  • The box model is css property.Using this we give some padding, margin, border to any div or container. It consist of content,padding,border and margin.

    • What is difference padding and margin?

    Padding Margin
    Padding is used inside the box and it is space between content and border. Margin is used outside the box and it is space between border and near by elements.
    Padding only gives positive value. Margin gives both positive and negative values.
    If we increase padding value then box size also increase. If we increase margin value then space between box and near by element increases.

    Difference between flex row and column.

    Flex row Flex Column
    Flex row is sub property of display flex.In which using flex direction row we arrange flex items in horizontal direction. Flex column is also sub property of display flex.In which by using flex direction column we arrange flex items in vertical direction or column wise.
    Main axis is X-axis and cross axis is Y-axis. Main axis is Y-axis and cross axis is X-axis.

    Justify-content allow you to do what?

  • Justify content is sub property of display flex and it has six sub property.By using justify content we arrange flex items at start of flex container or centre or end accordingly. The main axis is X-axis and it has six sub properties as follows:
  • flex-start
  • flex-end
  • center
  • space-around
  • space-evenly
  • space-around

    • Difference between absolute and relative position

    Absolute Position Relative Position
    When we give position absolute to any elements it moves according to its parent element. When we give position relative to any element it moves according to its near by elements.
    In absence of parent element it moves according to body element. When we set some left,right,top and bottom propeties it moves according to its normal position.

    What is z-index in CSS?

  • z-index property determines stack level of HTML elements. The stack level refers to an element's position on z-axis. Element with higher z-index value will be placed on top of element with lower z-index value.Z-index only work when elements position has been set to any one of the position.
